# Raw PDF Placement

Place the official ECT PDFs for Chaiyaphum constituency 2 in these folders:

- `5_16/`
- `5_16_partylist/`
- `5_17/`
- `5_17_partylist/`
- `5_18/`
- `5_18_partylist/`

The expected file names are listed in `configs/chaiyaphum_2_manifest.csv`.
If the downloaded ECT files have different names, either rename them to match
the manifest or update the manifest `file_path` values.
